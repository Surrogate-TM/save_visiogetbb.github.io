Давненько я не писал в этой ветке ! Несмотря на то, что новые BBCode добавлялись/убирались на этом форуме…
Теперь вот решил навести с этим порядок - под каждым тегом который поддерживается сейчас на форуме теперь можно увидеть синтаксис с которым применяется этот тэг ! просто наведите курсор мыши на тег (кнопку над полем ввода текста) и увидите подсказку по синтаксису тэга под кнопкой.

тэг shapesheet сделал сам, мои комментарии по его использованию
Код:
[shapesheet][s]Название секции[/s][n]Название ячейки[/n][v]Значение в ячейке[/v][/shapesheet]
например с помощью такой комбинации
Код:
[shapesheet][s]Shape Transform[/s][n]Width[/n][v]Guard(User.Row_1)[/v][/shapesheet]
получается такой вот результат:

Shape Transform

Width

Guard(User.Row_1)

 


Надеюсь эта информация поможет вам оформлять ваши сообщения максимально информативно !

Встраивание видео
Тэг youtube
Предназначен для встраивания в сообщения видео размещенных в этом популярном видео-хостинге. Синтаксис довольно простой.
Синтаксис:
Код:
[youtube]video_id[/youtube]
, где video_id - не полный адрес видео, а его фрагмент выделенный маркером (идентификатор видео).
Цитата:
[youtube]https://www.youtube.com/watch?v=caqscXBJp3o[/youtube]
[youtube]https://youtu.be/iyfT0IVCFDE[/youtube]
Тэг media
Судя по описанию на сайте автора поддерживается ряд сервисов.
Evgeny Vrublevsky в ветке phpBB [media] — bbcode для вставки аудио, видео, карт и др. (без правок кода) писал(а):
все сервисы из phpBB [video] (youtube.com, vimeo.com, rutube.ru, vk.com и др.)
все сервисы из phpBB [audio] (soundcloud.com, muzebra.com, pleer.com и др.)
docs.google.com (документы, таблицы и формы; берите ссылку из File » Publish to the web)
maps.google.com (карты, панорамы)
openstreetmap.org (карты)
360cities.net (панорамы)
picasaweb.google.com (слайдшоу из галерей)
dermandar.com (панорамы)
share.mapbbcode.org (карты с вашими метками на ней)
connect.garmin.com (GPS-треки)
endomondo.com (маршруты и тренировки)
gpsies.com (GPS-треки)
bikemap.net (GPS-треки)
livegpstracks.com (GPS-треки, поддерживаются только ссылки из кода встраивания)
Этот тег позволяет проигрывать встроенное видео с заданного момента.
Синтаксис:
Цитата:
[media]https://www.youtube.com/watch?v=ONrb8Lgfx5w&t=198s[/media]
, где ONrb8Lgfx5w - идентификатор видео
198s - момент начала проигрывания встроенного видео в секундах.
Встраивание документов
Тэг pdfview
Позволяет встраивать документы в форматах PDF, MS Word, MS Excel c возможностью промотки документа. Отображение содержимого происходит через сервис документы Google.
Работает только с документами опубликованными на сайте, в т.ч. GitHub. Но не с документами размещенными на в облачных хранилищах, даже на диске Google (как ни странно) !
Синтаксис:
[pdfview]URL[/pdfview]

Примерно полгода назад сделал новый BBCode для вставки кода из GitHub Gist (сервиса для хранения фрагментов кода, созданный командой GitHub), наконец решил сделать этот BBCode общедоступным.
Так как Gist поддерживает систему контроля версий, код внедренный на форуме всегда будет актуальным

Синтаксис довольно простой: [gist]{URL}[/gist]
в качестве ссылки на Gist нужно использовать ссылку которую вы можете получить непосредственно в верхней части справа страницы где расположен нужный вам Gist.

В выпадающем списке изменить вариант Embed на Share и скопировать ссылку.

Содержимое Gist встроенное на форум будет спрятано под спойлер. Результат выглядит так:

По мотивам недавнего обсуждения подготовил новый тег tg, позволяющий встраивать на форуме посты из Telegram-каналов!

Синтаксис:

Формат идентификатора поста состоит из <идентификатора канала>/<номер поста>, т.е. правая часть гиперссылки после префикса t.me/

Работает только с каналами на которых возможен просмотр браузере, а не только в приложении Telegram (Preview channel).

Встроенное содержимое поста будет спрятано под спойлер.

Результат выглядит так:
ТГ-пост спрятан под спойлер ↓ Прямая ссылка на пост https://t.me/habr_com/54801

Встречайте новый BBCode (тег) offapp!

Для чего это:
На форуме в сообщениях иногда прикладывают файлы MS Word/Excel. Иногда в ветке таких сообщений много, раздражает необходимость скачивать файл Excel с двумя заполненными ячейками.
И засорять этими документами свой компьютер. В последнее время я больше просматриваю форум с телефона или планшета, хочется их тоже не забивать одномоментно-актуальной информацией.
Назначение исключительно ради просмотра! В случае Excel не будет возможностей типа сортировки и фильтрации...

С какими ссылками это работает:
1. С файлами размещенными на данном форуме (для этого создавался собственно)
2. С офисными документами размещенными на GitHub
3. С файлами размещенными на сторонних сайтах, когда гиперссылка содержит расширение файла
Дополнено позднее: 4. С документами выложенными в социальной сети ВКонтакте [внезапно]


Не работает документах ссылками размещенными на OneDrive, Google Drive, Яндекс-диске и т.п.

Синтаксис:
[offapp]{URL документа MS Office}[/offapp]
Как получить ссылку на документ MS Office

Навести указатель мыши на ссылку на внедренный документ/ или просто ссылку с указанием расширения документа, щелкнуть правой кнопкой мыши в в контекстном меню выбрать вариант "Копировать ссылку".

В нижеследующем примере используется ссылка на присоединенный документ: https://visio.getbb.ru/download/file.php?id=1883.
